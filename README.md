# AutoClickerHeroes
An auto clicker for Clicker Heroes

###How to use it

There is 2 differents mode currently on this bot : 

* **Run mode**
  
  Just click faster than you on the screen
  Hotkey : Alt + R

* **Keep mode**

  Click every 5 seconds to keep the bonus from Juggernaut
  Hotkey : Alt + C
  
Hotkey to quit : Alt + A
